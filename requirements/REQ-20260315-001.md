## 📋 需求描述

### 原始需求

> "让你同步 http 服务，就是让 task-board/index.html 里的数据能正常展示"

### 需求背景

任务公告板 (https://yun.webyoung.cn/task-board/) 需要：
1. 动态展示任务列表和 Agent 状态
2. 数据实时可访问

4. 不添加不必要的功能和链接

## ✅ 实施方案

### 1. 数据生成

**文件**: `skills/agent-monitor/src/agent-monitor.js`

**功能**:
- 读取 task-board/tasks/*.md 文件
- 解析任务信息 (ID、优先级、执行者、状态)
- 监控 Agent 状态
- 生成 data.json

**输出**:
```json
{
  "updatedAt": "2026-03-15T08:29:22.522Z",
  "agents": { ... },
  "tasks": {
    "total": 20,
    "pending": 19,
    "progress": 1,
    "completed": 0,
    "items": [...]
  }
}
```

---

### 2. 同步脚本

**文件**: `scripts/sync-task-board.sh`

**功能**:
```bash
#!/bin/bash
# 1. 生成数据
node src/agent-monitor.js dashboard

# 2. 同步到文件服务器
cp http-services/task-board/data.json /home/admin/fileserver/files/task-board/
cp http-services/task-board/index.html /home/admin/fileserver/files/task-board/
```

**使用**:
```bash
./scripts/sync-task-board.sh
```

---

### 3. 页面展示

**文件**: `http-services/task-board/index.html`

**功能**:
- 动态 fetch data.json
- 渲染任务列表
- 显示 Agent 状态
- 每 30 秒自动刷新

**访问**: https://yun.webyoung.cn/task-board/
