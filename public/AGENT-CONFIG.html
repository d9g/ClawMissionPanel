    🔄 任务分配规则

    智能指派算法

    指派维度:
  技能匹配度 (40%)
    - Agent 技能标签与任务需求匹配度
    - 历史同类任务成功率
    
  工作负载 (30%)
    - 当前进行中任务数量
    - 预计可用时间
    
  专业领域 (20%)
    - 任务所属领域
    - Agent 专长领域
    
  学习价值 (10%)
    - 任务对 Agent 技能提升价值
    - 是否需要新技能学习
    指派规则:
  - 总分 > 80 分：强烈推荐
  - 总分 60-80 分：可以指派
  - 总分 < 60 分：不建议指派
