    恢复流程

    方式 A：完整恢复（推荐）

    1. 找到最新备份
    
cd /home/admin/.openclaw/workspace/person/xiaoyun/backups/
ls -la xiaoyun/
    2. 执行恢复
    
./scripts/restore.sh /home/admin/.openclaw/workspace/person/xiaoyun/backups/xiaoyun/20260311-184500
    3. 验证恢复
    
openclaw agents list
openclaw skills list
    4. 重启 Gateway
    
openclaw gateway restart

    方式 B：手动恢复特定文件

    恢复 Agent 配置
    
cp -r /path/to/backup/agents/* ~/.openclaw/agents/
    恢复记忆文件
    
cp -r /path/to/backup/workspace/memory/* ~/workspace/memory/
    恢复技能文件
    
cp -r /path/to/backup/workspace/skills/* ~/workspace/skills/

    跨机器恢复

    场景：在新机器上 Clone 一个一模一样的团队

    前提条件

  • 新机器已安装 OpenClaw
  • 已配置相同的认证信息（API Key 等）
  • 可访问备份文件（Git 仓库或文件传输）

    • 恢复步骤

    1. 克隆备份仓库（或传输备份文件）
    
git clone git@github.com:d9g/person.git /tmp/openclaw-backup
    2. 找到最新备份
    
ls -la /tmp/openclaw-backup/xiaoyun/backups/xiaoyun/
    3. 执行恢复脚本
    
bash /tmp/openclaw-backup/xiaoyun/scripts/restore.sh \
  /tmp/openclaw-backup/xiaoyun/backups/xiaoyun/20260311-184500
    4. 验证 Agent 配置
    
openclaw agents list

    应该看到：
    

    - main (小云)
    

    - xiaoyun-novel (推书酱)
    

    - xiaoyun-judge (小云评委)
    

    - xiaoyun-recorder (小云记录)
    

    - xiaoyun-dev (小云开发)
    5. 验证技能配置
    
openclaw skills list
    6. 验证 HTTP 文档
    
curl -I https://yun.webyoung.cn/task-board/SYSTEM-DESIGN.html
    7. 重启 Gateway
    
openclaw gateway restart

    常见问题

    Q1: 恢复后 Agent 不显示？

    A: 检查 agents 目录权限和 openclaw.json 配置 

    ls -la ~/.openclaw/agents/
cat ~/.openclaw/openclaw.json | grep -A10 '"agents"'

    Q2: 恢复后 HTTP 文档 404？

    A: 文件未复制到 fileserver 目录 

    手动复制
    
cp ~/workspace/task-board/*.html /home/admin/fileserver/files/task-board/
chmod 644 /home/admin/fileserver/files/task-board/*.html

    Q3: 跨机器恢复后 API Key 丢失？

    A: API Key 存储在 auth-profiles.json，需要单独配置 

    方式 1: 从备份恢复
    
cp /path/to/backup/agents/main/agent/auth-profiles.json \
   ~/.openclaw/agents/main/agent/auth-profiles.json
    方式 2: 重新配置
    
openclaw agents add <agent-id>

    Q4: 恢复后技能不工作？

    A: 检查技能目录和依赖 

    检查技能目录
    
ls ~/workspace/skills/
    重新安装依赖
    
cd ~/workspace/skills/<skill-name>/
npm install

    Q5: 如何只恢复特定 Agent？

    A: 手动复制特定 Agent 目录 

    只恢复小云评委
    
cp -r /path/to/backup/agents/xiaoyun-judge \
      ~/.openclaw/agents/xiaoyun-judge
    只恢复推书酱
    
cp -r /path/to/backup/agents/xiaoyun-novel \
      ~/.openclaw/agents/xiaoyun-novel
