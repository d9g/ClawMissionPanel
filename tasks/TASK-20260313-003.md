3. 验收 API
4. 验收记录

---

## 📊 功能需求

### 1. 验收状态

**新增状态**:
- `PENDING_ACCEPTANCE` - 待验收
- `ACCEPTING` - 验收中
- `ACCEPTED` - 验收通过
- `REJECTED` - 验收驳回

**状态流转**:
```
IN_PROGRESS → PENDING_ACCEPTANCE → ACCEPTING → ACCEPTED → COMPLETED
                                                      ↓
                                                 REJECTED
```

### 2. 验收页面

**URL**: `/task-board/acceptance/TASK-XXX.html`

**功能**:
- 显示任务详情
- 显示交付物
- 验收标准检查清单
- 验收意见输入
- 通过/驳回按钮
- 验收历史记录

### 3. 验收 API

**接口**:
```javascript
POST /api/tasks/:taskId/acceptance/start
  - 开始验收
  - 状态：PENDING_ACCEPTANCE → ACCEPTING

POST /api/tasks/:taskId/acceptance/accept
  - 验收通过
  - 状态：ACCEPTING → ACCEPTED
  - 参数：comments, acceptor

POST /api/tasks/:taskId/acceptance/reject
  - 验收驳回
  - 状态：ACCEPTING → REJECTED
  - 参数：reasons, acceptor, suggestions
```

### 4. 验收记录

**数据表**: `task_acceptances`

**字段**:
- id
- task_id
- acceptor
- result (accepted/rejected)
- comments
- reasons
- suggestions
- created_at
