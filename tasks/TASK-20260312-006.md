在任务公告板首页增加 Agent 工作状态展示，让用户无需询问即可看到所有 Agent 的进度。

## 🎯 需求说明

### 5. 自动刷新功能 ⬜️

**文件**: `public/js/dashboard.js` (新建)

**功能**:
- 每 5 分钟自动刷新数据
- 显示最后更新时间
- 错误处理

**代码示例**:
```javascript
// 每 5 分钟刷新一次
setInterval(() => {
  fetch('/api/agents/status')
    .then(r => r.json())
    .then(data => updateAgentCards(data))
    .catch(err => console.error('刷新失败:', err));
}, 300000);

// 显示最后更新时间
function updateLastUpdated() {
  const now = new Date();
  document.getElementById('last-updated').textContent = 
    `最后更新：${now.getHours().toString().padStart(2,'0')}:${now.getMinutes().toString().padStart(2,'0')}`;
}
```
