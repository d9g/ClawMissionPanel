## 📊 功能需求

### 1. 依赖字段

**任务格式新增字段**:
```markdown
**依赖任务**: TASK-XXX, TASK-YYY
**被依赖**: TASK-ZZZ
```

### 2. 依赖解析

**功能**:
- 解析任务文件中的依赖关系
- 构建依赖图
- 检测循环依赖

### 3. 依赖阻塞检测

**规则**:
- 依赖任务未完成 → 当前任务 BLOCKED
- 依赖任务延期 → 当前任务风险告警

### 4. 依赖可视化

**详情页显示**:
- 依赖任务列表
- 被依赖任务列表
- 依赖状态（已完成/进行中/未开始）
