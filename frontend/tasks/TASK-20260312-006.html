📋 任务目标

在任务公告板首页增加 Agent 工作状态展示，让用户无需询问即可看到所有 Agent 的进度。

---

🎯 需求说明

### 用户原话

"这个信息最好也能在任务公告面板里体现。要不然每次都得我给你发信息，询问，你有时候也挺忙的。"

### 核心需求

1. 在任务板首页展示所有 Agent 的工作状态

2. 显示任务进度百分比

3. 延期任务有明显告警标记

4. 数据自动刷新 (每 5 分钟)

---

📋 功能拆解

### 1. Agent 状态 API ⬜️

**端点**: `GET /api/agents/status`

**功能**:

  • 查询所有已配置 Agent
  • 获取每个 Agent 的当前任务
  • 计算任务进度
  • 检测是否延期

    • **响应示例**:

    ```json

    {

    "success": true,

    "data": [

    {

    "id": "xiaoyun-dev",

    "name": "小云开发",

    "emoji": "💻",

    "status": "working",

    "currentTask": "TASK-20260312-001",

    "progress": 85,

    "overdue": false

    },

    {

    "id": "xiaoyun-test",

    "name": "小云测试",

    "emoji": "✅",

    "status": "inactive",

    "currentTask": "TASK-20260311-004",

    "progress": 15,

    "overdue": true,

    "overdueHours": 9

    }

    ]

    }

    ```

    **预计时间**: 1h

    ---

    ### 2. 进度统计 API ⬜️

    **端点**: `GET /api/tasks/stats`

    **功能**:

  • 统计任务总数
  • 统计各状态任务数
  • 统计延期任务数
  • 计算完成率

    • **响应示例**:

    ```json

    {

    "success": true,

    "data": {

    "total": 7,

    "completed": 2,

    "inProgress": 3,

    "pending": 2,

    "overdue": 2,

    "lastUpdated": "2026-03-12T16:00:00Z"

    }

    }

    ```

    **预计时间**: 0.5h

    ### 5. 自动刷新功能 ⬜️

    **文件**: `public/js/dashboard.js` (新建)

    **功能**:

  • 每 5 分钟自动刷新数据
  • 显示最后更新时间
  • 错误处理

    • **代码示例**:

    ```javascript

    // 每 5 分钟刷新一次

    setInterval(() => {

    fetch('/api/agents/status')

    .then(r => r.json())

    .then(data => updateAgentCards(data))

    .catch(err => console.error('刷新失败:', err));

    }, 300000);

    // 显示最后更新时间

    function updateLastUpdated() {

    const now = new Date();

    document.getElementById('last-updated').textContent =

    `最后更新：${now.getHours().toString().padStart(2,'0')}:${now.getMinutes().toString().padStart(2,'0')}`;

    }

    ```
